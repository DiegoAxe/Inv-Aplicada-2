Almacenamiento: Indexed DB

Conceptos, e información básica de IndexedDB

IndexedDB es una forma de almacenar datos de manera persistente, dentro de nuestro navegador, debido a que permite crear aplicaciones con capacidades de consulta enriquecidas, puede funcionar tanto si tenemos acceso a internet, como si no lo tenemos, lo que es una gran ventaja.


Este es un sistema de base de datos transaccional, el cual está basado en SQL, sin embargo, indexedDB es una base de datos orientada a objetos en JavaScript, la cual permite almacenar, y recuperar objetos que están indexados, por medio de una clave.


Es ideal para almacenar grandes cantidades de datos las cuales requieren ser capaces de ser búscadas rapidamente, en sitios tales como, un catalogo de productos, registros, es completamente ideal.


Algunas de las ventajas de IndexedDB son las siguientes:


  1. Almacena casi todo tipo de valores por claves.
  2. Soporta transacciones para mayor fiabilidad.
  3. Soporta consultas de rango por clave, e índices.
  4. Puede almacenar mucho mayor volumen de datos que el localStorage.
