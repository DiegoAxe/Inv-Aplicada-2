Almacenamiento: SessionStorage

Conceptos, e información básica de SessionStorage

SessionStorage: al contrario del local, este solamente se mantiene asociado a la sesión actual, la información perdura mientras el navegador se encuentre abierto, momento de cerrar el navegador se pierde, el sessionStorage soporta las recargas de página, o reaperturas de la página.


Si se abre una nueva pestaña o ventana iniciara como nueva sesión, lo cual es lo contrario a cómo trabajan las cookies.


La sintaxis del sessionStorage es la siguiente:


Almacena la información en sessionStorage: sessionStorage.setItem('ñ', 'value');


Obtiene la información almacenada desde sessionStorage: var data = sessionStorage.getItem('ñ');



Ejemplo práctico - Contador

Contador:

0

Presione el botón para que el contador incremente. No importa si se recarga la página.*

*Al recargar la página, el contador regresa a 0, pero este guarda los datos con los que se había quedado antes de recargarla, por lo tanto, al presionar de nuevo el botón, este inicia desde el último dato registrado.
